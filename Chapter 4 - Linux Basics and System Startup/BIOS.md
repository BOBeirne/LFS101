- BIOS - **B**asic **I**nput / **O**utput **S**ystem
- Performs [[POST]] test (Power On Self test)
	- Initializes the screen and keyboard
	- tests the main memory
- BIOS software is stored in ROM (read-only-memory) chip on the motherboard
- After this step, the rest of the [[BOOT]] is controlled by OS


### BIOS Boot sequence

1) BIOS
2) [[CMOS]]
3) [[MBR]]
4) [[BOOT Loader]]
5) [[Kernel]]
6) [[OS]]
