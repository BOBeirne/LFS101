
### Downloading Image

- You can either do internet download via Hyper-V quick create or download manually
- Manual download:
1) go to [download link](https://ubuntu.com/download/desktop)
2) choose Ubuntu 24.04.3 LTS (or latest)
3) Download the iso file
4) Mount into VM

### Installation (Hyper-V)

1) Quick create...
2) Select either the Ubuntu listed in the side menu for network download or the Local installation source if you downloaded `.iso` manually
3) ISO: 
	1) tick OFF the box "this vm will run windows"
	2) Change install source -> Select ISO

4) Let the system download or / and install 
5) Select language and keyboard layout
6) Select **Normal Installation**
7) Tick **download updates** while installing (for newest updates)
8) Tick **install 3rd party software** for gpu, wifi hardware etc (drivers)
9) Pick drive or partition to use
10) Select location
11) Create **user** + **password** + (optional) change system name
12) tick **require password to login**
13) Let the system download or / and install 
14) **restart** once prompted
### Post-installation Setup

1) (optional) login to account
2) (optional) send system info to Canonical (ubuntu owners)
3) Privacy options - location services
4) Install optional software
