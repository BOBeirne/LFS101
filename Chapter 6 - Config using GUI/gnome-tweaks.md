
- You can access it by pressing `ALT-F2` and typing in `gnome-extensions-app`, it may be called `gnome-tweak-tool` in older Linux versions

- It provides much more granular access to [[GNOME]] [[GUI]] appearance and behaviour
- It also allows to install [[Extensions]], although some distros has placed them in separate utility called `gnome-extensions-app`
- It may not be installed by default but it can be installed
