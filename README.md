# LFS101 - Introduction to Linux

Personal study notes for [LFS101](https://training.linuxfoundation.org/training/introduction-to-linux/) by The Linux Foundation.

## Notes

All notes are written in Markdown for use with [Obsidian](https://obsidian.md/). 

Content derives from the LFS101 course materials with occasional supplementary notes from hands-on practice.

## Structure

- **Chapter 1** - Course Information
- **Chapter 2** - Linux Foundation and main distribution families: Debian, Red Hat, SUSE
- **Chapter 3** - Linux Philosophy and Concepts (History, kernel, licensing, terminology)
- **Chapter 4** - Basics and System Startup Process
- **Chapter 5** - Graphical User Interface
- **Chapter 6** - System configuration using GUI
- **Chapter 7** - Common Applications 
- **Chapter 8** - CLI operations

*More chapters will be added as I progress through the course.*
