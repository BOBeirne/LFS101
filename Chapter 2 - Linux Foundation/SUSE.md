## The SUSE Family

 `S.u.S.E.` - Software und System-Entwicklung. 
 German for **"Software and System Development"** Founded in Germany in 1994.
 
- Consists of SLES (SUSE Linux Enterprise Server) and OpenSUSE
- similar relationship as in Red Hat family
- uses the RPM-based `zypper` package manager to install, update, and remove packages in the system.
- It includes the `YaST` (Yet Another Setup Tool) application for **system administration** purposes.

### openSUSE

- Used as reference distro for SUSE family
- Free to use
- Comes in 2 flavours:
	- `openSUSE Tumbleweed` - rolling release, always the latest packages (like Fedora's pace)
		- more independent and can contribute innovations back to `SLES`
		- For Developers, openSUSE Contributors, Gamers and Linux/FOSS Enthusiasts
	- `openSUSE Leap` - regular releases, shares codebase with `SLES` **(more stable, closer to the enterprise product)**
		- For Sysadmins, Enterprise Developers, and ‘Regular’ Desktop Users

### SLES

- Linux Enterprise Server
- upstream for openSUSE
- widely used in retail and many other sectors.
