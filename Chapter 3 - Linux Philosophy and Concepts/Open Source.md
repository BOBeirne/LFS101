
# Examples of popular OSS projects

- Android - built on top of Linux kernel
- httpd - Apache Web Server - about half of all webservers run Apache
- fb/twitter/ig all use OSS components
- Google search engine - and others too are running Linux
- Weather forecasting - all of world's supercomputers use Linux (one of those is weather forecasting)
- Fitbit and other fitness devices
- DVRs - almost all use Linux
- Medical devices - large number of them run embedded Linux
